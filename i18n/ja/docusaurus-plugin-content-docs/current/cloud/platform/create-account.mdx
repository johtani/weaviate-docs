Weaviate Cloud（ WCD ）にはインタラクティブ コンソールがあります。ユーザー アカウントを作成し、ログインして WCD クラスターの管理、クエリの実行、組織情報の設定を行いましょう。

## 新規ユーザー アカウントの作成

以下の手順で Weaviate Cloud アカウントを作成します。

## 計算・請求情報の設定

Sandbox インスタンスは、14 日で期限が切れる無償の短期インスタンスです。Weaviate を試すために Sandbox インスタンスを使用する場合、請求の設定は不要です。

Serverless クラスターは、運用・開発・テストに適した永続インスタンスです。Serverless クラスターは有料インスタンスで、料金はインスタンスタイプと選択したサポート プランによって異なります。Serverless インスタンスを作成する前に請求設定を行う必要があります。[こちらのガイド](/cloud/platform/billing)に従って請求アカウントを設定してください。

[AWS](https://aws.amazon.com/marketplace/pp/prodview-ng2dfhb4yjoic) や [GCP](https://console.cloud.google.com/marketplace/product/weaviate-gcp-mktplace/weaviate?inv=1&invt=AbwjlA) などのクラウド マーケットプレイス経由で登録した場合、請求はそのクラウド プロバイダーにまとめて行われます。
