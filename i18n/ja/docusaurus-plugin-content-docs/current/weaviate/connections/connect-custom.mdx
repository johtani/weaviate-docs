[ Python Client v4 ](/weaviate/client-libraries/python) と [ TypeScript Client v3 ](../client-libraries/typescript/index.mdx) には、一般的な接続タイプ向けのヘルパーメソッドが用意されています。さらに、追加の接続設定が必要な場合のためにカスタムメソッドも提供しています。

その他のクライアントをお使いの場合でも、標準の接続メソッドはすべての接続で設定可能です。
