## Weaviate とは

Weaviate は、データオブジェクトとそのベクトル埋め込みの両方を保存・インデックスできる **オープンソースのベクトルデータベース** です。キーワードの一致だけでなく、ベクトルにエンコードされた意味を比較することで、高度なセマンティック検索を実現します。主な機能は次のとおりです。

- **[セマンティック検索とハイブリッド検索](./search/basics.md)**  
  データをベクトルでインデックス化することで、意味的類似度とキーワードの両方に基づく検索をサポートします。これにより、クエリが保存されたデータと完全に一致しなくても、より関連性の高い結果を得られます。

- **[検索拡張生成 (RAG)](./search/generative.md)**  
  Weaviate は RAG ワークフローの堅牢なバックエンドとして機能できます。ベクトル検索で取得したコンテキストを生成モデルに提供し、正確で文脈に即した回答生成を支援します。

- **[エージェント駆動ワークフロー](../agents/index.md)**  
  柔軟な API と最新の AI モデルとの統合により、インテリジェントなエージェントを活用するアプリケーションの基盤として適しています。エージェントは、Weaviate に保存されたデータからセマンティックな洞察を得て意思決定やアクションを実行できます。

## Weaviate エコシステム

Weaviate エコシステムは、クラウドネイティブで AI を活用したアプリケーション構築を中心とした複数のツールとサービスで構成されています。

上記のハイレベルな概要図のとおり、エコシステムは次の要素で構成されています。

- **[Weaviate Database](#what-is-weaviate)**: オブジェクトとベクトルを保存するオープンソースのベクトルデータベース。
- **[Weaviate Cloud](/cloud)**: Weaviate ベクトルデータベースのフルマネージドクラウドデプロイ。
- **[Weaviate Agents](/agents)**: Weaviate Cloud ユーザー向けのプリビルトなエージェントサービス。
- **[Weaviate Embeddings](/cloud/embeddings)**: Weaviate Cloud ユーザー向けのマネージド埋め込み推論サービス。
- **[外部モデルプロバイダー](/weaviate/model-providers)**: Weaviate と統合可能なサードパーティモデル。
